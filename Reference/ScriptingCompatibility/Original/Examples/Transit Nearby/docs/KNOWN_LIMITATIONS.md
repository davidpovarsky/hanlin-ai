# מגבלות ידועות

- תכנון מסע, חיפוש תחנות אמיתי לפי רדיוס וגאומטריית מסלול של BusNearby דורשים אסימון Bearer תקף. אין בלכידות זרימת הנפקה מתועדת ובטוחה, ולכן ניתן להזין אסימון ידנית בלבד.
- ללא אסימון, האפליקציה נשארת שימושית דרך חיפוש התחנות הציבורי ו־KavNav: לוחות זמנים, זמן אמת, התראות, קווים ורכבים. איתור קרוב מבצע fallback לפי כתובת שהמערכת מפיקה מהמיקום.
- KavNav אינו מחזיר בלכידה פוליליין מוכן לציור. לכן ללא אסימון BusNearby ייתכן שמפת קו תציג תחנות ורכבים בלי קו מחבר.
- סכימת התראות BusNearby לא הופיעה בתשובות הלא־ריקות שנלכדו; התראות KavNav הן המקור הראשי.
- לא ניתן לבצע כאן בדיקת UI אמיתית של הרשאות מיקום, Dynamic Type, VoiceOver, iPad Split View, Widget ו־Live Activity. נדרשת בדיקה במכשיר iPhone/iPad בתוך Scripting.
- תזמון Widget ו־Live Activity כפוף למדיניות iOS ואינו מבטיח רענון רשת בתדירות המבוקשת.
