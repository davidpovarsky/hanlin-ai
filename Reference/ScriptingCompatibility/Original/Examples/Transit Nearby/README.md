# Transit Nearby — תחבורה קרובה

אפליקציית תחבורה ציבורית ישראלית, נייטיבית ל־Scripting, ב־TypeScript/TSX. הממשק עברי ו־RTL ומשתמש ברכיבי Scripting בלבד: ניווט, רשימות, טפסים, MapKit, Widget ו־Live Activity.

## הפעלה

1. פתח את התיקייה `Transit Nearby` באפליקציית Scripting.
2. הפעל את `index.tsx`.
3. במסך ״קרוב״ בחר אם להשתמש במיקום הנוכחי, לבחור מקום במפה או לחפש תחנה ידנית.
4. לבדיקת TypeScript מהמחשב, הרץ משורש סביבת העבודה:

   ```text
   npm run typecheck
   ```

## יכולות

- תחנות קרובות עם מרחק, זמן הליכה משוער ושלוש הגעות קרובות.
- חיפוש ציבורי לפי שם תחנה, רחוב, קוד תחנה או מספר קו.
- לוח תחנה שמשלב לוח זמנים ונתוני זמן אמת, מציג שעה מדויקת ומסמן מידע מתעכב או מיושן.
- התראות שירות של KavNav.
- פרטי קו עם סדר תחנות, מגבלות איסוף/הורדה, לוח יציאות, רכבים פעילים ומפת מסלול כשהגאומטריה זמינה.
- מועדפים נפרדים לתחנות ולקווים, חיפושים אחרונים ומטמון מקומי.
- תכנון מסע, כולל זמן יציאה/הגעה, נגישות, הליכה ושלבי מסע, כאשר קיים אסימון BusNearby.
- מפה נייטיבית עם תחנות, רכבים, בחירת תחנה וכרטיס הגעות.
- Widget בגדלי מערכת ומסך נעילה, App Intents ו־Live Activity למעקב אחר הגעה.
- מצבי טעינה, ריק, שגיאה, מידע חלקי ומידע שמור/מיושן.

## מקורות נתונים

- **KavNav** — המקור הראשי ללוחות זמנים, זמן אמת, מטא־דאטה של קווים והתראות.
- **BusNearby** — חיפוש תחנות וקווים ציבורי; תחנות לפי רדיוס, גאומטריית קו ותכנון מסע במצב מאומת.
- **Apple MapKit / Location** — מפה, בחירת מקום, מיקום וגאוקודינג מערכת.

האפליקציה אינה קשורה לבעלי השירותים ואינה טוענת לבעלות על הנתונים.

## אימות BusNearby

בלכידות שסופקו לא הופיעה זרימת הנפקת אסימון יציבה שאפשר לממש ללא ניחוש. לכן ניתן להזין אסימון Bearer ידנית במסך הגדרות. האסימון:

- נשמר רק ב־Keychain המבודד של הפרויקט;
- אינו נכתב ל־Storage, למטמון או ללוגים;
- מוסר אוטומטית לאחר תשובת `401`/`403`;
- לעולם אינו נכלל בקובצי המקור.

ללא אסימון עדיין פועלים חיפוש תחנות וקווים, לוחות זמנים, זמן אמת, התראות, מועדפים, מפות תחנות ורכבים. תכנון מסע וגאומטריית קו עשויים לא להיות זמינים.

## iPhone ו־iPad

- ב־iPhone יש TabView נייטיבי: קרוב, חיפוש, מועדפים, מפה והגדרות. לכל אזור NavigationStack משלו.
- ב־iPad יש NavigationSplitView: סרגל צד ליעדים הראשיים, עמודת תוכן ומפת פרטים קבועה. במסך קו מוצגים ציר התחנות והמפה זה לצד זה.
- הפריסה משתמשת ב־Dynamic Type, רכיבי מערכת ו־RTL; מספרי קווים ושעות נשארים בתצוגה מספרית יציבה.

## מבנה

- `index.tsx` — נקודת הכניסה הראשית.
- `src/app` — מעטפת iPhone/iPad וניווט.
- `src/data` — חוזי API גולמיים, HTTP, לקוחות, אימות ומאגר מנורמל.
- `src/domain` — מודלים שאינם תלויים בתשובות הגולמיות.
- `src/storage` — מועדפים, חיפושים, העדפות ומטמון עם TTL.
- `src/screens` — מסכי קרבה, חיפוש, תחנה, קו, מסע, מפה, מועדפים והגדרות.
- `src/components` — תגי קו, כרטיסי תחנה, שורות הגעה והתראות.
- `widget.tsx`, `live_activity.tsx`, `app_intents.tsx` — הרחבות מערכת.
- `docs/` — חוזי API, ארכיטקטורה ומגבלות ידועות.
